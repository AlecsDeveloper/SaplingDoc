Módulo Entity


¿Qué es?


El módulo Entity es el encargado de la manipulación de entidades del mundo, siendo este el responsable de agregar funciones a entidades dentro del juego.



Comandos


Los comandos disponibles en el módulo TNT son los siguientes:




Cada comando puede habilitar o deshabilitar una función del módulo.



Funciones


Guardian Drop Sponges


La función "guardianDropSponges" permite regenerar esponjas a manera de drop mediante guardianes. Esto evita depender de las esponjas naturales limitadas.


Husk Drop Sand


La función "huskDropSand" permite regenerar arena a manera de drop mediante momias. Esto permite la creación de nuevas granjas dentro del juego.


Ghast Drop Quartz


La función "ghastDropQuartz" permite regenerar cuarzo a manera de drop mediante fantasmas. Esto permite la creación de nuevas granjas dentro del juego.


Silverfish Drop Gravel


La función "silverfishDropGravel" permite regenerar grava a manera de drop mediante peces plateados. Esto permite la creación de nuevas granjas dentro del juego.


Pigmans Farm Warts


En Minecraft Bedrock no es posible automatizar la plantación y recoleccion de verrugas del nether.

La finalidad de esta función es hacer posible dicha automatización mediante pigmans (Hombres cerdos zombificados) permitiendo que dichas entidades tengan la capacidad de recolectar y replantar las verrugas.
